JavaScript is supported by all modern websites. Basically, HTML is the structure, CSS makes it look pretty and JavaScript make it

Question: Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.

Answer: Web browsers and web servers function together as a client-server system. Web browsers and servers communicate using TCP/IP. Hypertext Transfer Protocol is the standard application protocol on top of TCP/IP supporting web browser requests and server responses.
