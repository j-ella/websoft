Question: Explain the role of HTML, CSS and JavaScript in web development.

Answer: HTML provides the basic stucture of an webpage, it is at the core of every webpage. CSS is used for the formatting, presentation and the style, for example type of the font, color, size and layout. JavaScript is used to control the behaviour of the elements. JavaScript is supported by all modern websites. Basically, HTML is the structure, CSS makes it look pretty and JavaScript make it

Question: Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.

Answer: Web browsers and web servers function together as a client-server system. Web browsers and servers communicate using TCP/IP. Hypertext Transfer Protocol is the standard application protocol on top of TCP/IP supporting web browser requests and server responses.
